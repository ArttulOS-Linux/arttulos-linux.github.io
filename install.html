Complete Installation Guide

Follow this comprehensive guide to install ArttulOS Linux on your computer. This guide works for beginners and advanced users alike.

Step 1: Download ArttulOS ISO

Visit our download page and get the latest ArttulOS ISO file.

  • File size: ~2.5 GB
  • Architecture: x86_64 (64-bit)
  • Desktop: Cinnamon

Step 2: Create Bootable USB Drive

Required: USB drive with at least 4GB capacity

On Linux:

# Replace /dev/sdX with your USB device (check with lsblk)
sudo dd if=arttulos-*.iso of=/dev/sdX bs=4M status=progress && sync

On Windows:

Use Rufus or Balena Etcher

  1. Download and run Rufus
  2. Select your USB drive
  3. Select the ArttulOS ISO file
  4. Click "START" and wait for completion

On macOS:

Use Balena Etcher (easiest) or Terminal:

# Find your USB device
diskutil list

# Unmount it (replace diskN with your device)
diskutil unmountDisk /dev/diskN

# Write ISO
sudo dd if=arttulos-*.iso of=/dev/rdiskN bs=1m

Step 3: Boot from USB

  1. Insert the USB drive into your computer
  2. Restart your computer
  3. Press the boot menu key during startup:
    • Dell: F12
    • HP: F9 or Esc
    • Lenovo: F12 or F8
    • Asus: F8 or Esc
    • Acer: F12
    • MSI: F11
    • Generic: F2, F10, F11, F12, Del, or Esc
  4. Select your USB drive from the boot menu
  5. ArttulOS will boot into the live environment

Step 4: Start the Installer

Once booted into ArttulOS live environment:

  1. Look for the "Install ArttulOS" icon on the desktop
  2. Double-click to launch the installer
  3. The Calamares installer will guide you through the process

Step 5: Configure Installation

Choose Your Language & Region

Select your preferred language, timezone, and keyboard layout.

Partition Your Drive

⚠ WARNING: Partitioning will erase data. Back up important files first!

Option 1: Erase Entire Disk (Recommended for beginners)

  • Select "Erase disk"
  • Choose your target drive
  • The installer will automatically create partitions

Option 2: Manual Partitioning (Advanced users)

  • EFI System Partition: 512 MB, FAT32, mounted at /boot/efi
  • Root Partition: 30+ GB, ext4, mounted at /
  • Home Partition (optional): Remaining space, ext4, mounted at /home
  • Swap (optional): 2-8 GB depending on RAM

Create User Account

  • Enter your full name
  • Choose a username (lowercase, no spaces)
  • Set a strong password
  • Choose whether to auto-login

Step 6: Install

  1. Review your installation summary
  2. Click "Install"
  3. Wait 5-15 minutes for installation to complete
  4. Installation will copy files, install bootloader, and configure system

Step 7: Reboot & Enjoy

  1. Click "Restart Now" when prompted
  2. Remove the USB drive when asked
  3. Your computer will boot into ArttulOS
  4. Log in with the credentials you created

Post-Installation Steps

Update Your System

sudo pacman -Syu

Install Additional Software

Use the built-in Pamac package manager or command line:

sudo pacman -S firefox vlc gimp libreoffice

Enable AUR (Arch User Repository)

AUR is already configured in ArttulOS! Just use Pamac GUI or install an AUR helper:

sudo pacman -S yay
yay -S google-chrome spotify
