Frequently Asked Questions

Can I run ArttulOS on old hardware?

Yes! ArttulOS runs on hardware from 2008+. As long as you have a 64-bit CPU and 2GB RAM, it will work. Performance will be best with 4GB+ RAM and an SSD.

Does ArttulOS support ARM processors (Raspberry Pi)?

Not yet. ArttulOS currently only supports x86_64 architecture. ARM support is on our roadmap for future releases.

How much RAM does ArttulOS really need?

The desktop environment uses ~700-900MB at idle. With Firefox and a few apps open, expect 2-3GB usage. We recommend 4GB minimum for a comfortable experience.

Can I dual boot with Windows?

Yes! ArttulOS installer (Calamares) detects Windows and can set up dual boot automatically. Just choose "Install alongside Windows" during partitioning.

Will my laptop's touchpad and function keys work?

Most modern laptops have excellent Linux support. Touchpads, keyboard backlights, brightness/volume keys typically work out of the box.
