01_SILICON_ARCHITECTURE

ArttulOS utilizes an x86_64-v3 instruction set baseline to ensure maximum performance across the RHEL binary shims and Arch rolling layers.

✓ Intel Core (v3+) ✓ AMD Ryzen / EPYC ✗ Legacy 32-bit (PURGED) ! ARM-64 (Experimental)

02_GRAPHICS_PIPELINE

Intel / AMD

Native open-source drivers integrated into the Arttul kernel shim. Mesa-standard.

NVIDIA

Proprietary RHEL-binary or Arch-DKMS modules enforced via PacNixum.

Acceleration

Full KVM/QEMU and VMware guest acceleration support for isolated node clusters.

03_ATOMIC_STORAGE_LOGIC

Due to the Nix Atomic Store architecture, ArttulOS maintains multiple system generations on-disk. We strictly enforce SSD storage. Traditional spindle HDDs will suffer critical timeouts during symlink resolution.
