General Questions

What is ArttulOS?
ArttulOS is a user-friendly Linux distribution based on Arch Linux. It combines the power and flexibility of Arch with an easy-to-use graphical installer and pre-configured Cinnamon desktop environment. Think of it as "Arch Linux made accessible" - you get rolling release updates, access to the AUR, and cutting-edge software without the complex manual installation process.
Is ArttulOS free?
Yes! ArttulOS is completely free and open source software licensed under the GNU GPL v3.0. You can use, modify, and distribute it freely. There are no hidden costs, subscriptions, or premium features.
How is ArttulOS different from Ubuntu or Linux Mint?
Key differences:
  • Rolling Release: ArttulOS gets continuous updates, while Ubuntu/Mint have fixed releases every 6-24 months
  • Package Base: Uses Arch repos + AUR (80,000+ packages) vs Debian/Ubuntu repos
  • Package Manager: Fast Pacman instead of APT
  • Philosophy: More customizable and closer to bleeding-edge software
  • Performance: Generally lighter and faster than Ubuntu
See our detailed comparison page for more info.
Is ArttulOS suitable for beginners?
Yes! While based on Arch Linux (known for being advanced), ArttulOS includes:
  • Graphical installer (no terminal commands needed)
  • Pre-configured desktop environment
  • GUI package manager (Pamac)
  • Common applications pre-installed
However, some basic Linux knowledge is helpful. If you're completely new to Linux, try the live USB first to get comfortable.
What does "rolling release" mean?
Rolling release means your system receives continuous updates without ever needing a major version upgrade. Run sudo pacman -Syu regularly and you'll always have the latest software. No need to reinstall when "ArttulOS 2.0" comes out - there won't be version numbers like that! Contrast with Ubuntu where you must upgrade from 22.04 → 24.04 → 26.04, etc.
Can I use ArttulOS for work/school?
Absolutely! ArttulOS includes or can easily install:
  • Office: LibreOffice (compatible with MS Office files)
  • Web: Firefox, Chrome, Brave
  • Communication: Discord, Slack, Teams (via web or flatpak)
  • Development: VS Code, JetBrains IDEs, Docker, Git
  • Media: GIMP, Inkscape, Kdenlive, OBS

Installation Questions

How do I install ArttulOS?
Follow our complete installation guide. In summary:
  1. Get the ISO from our homepage
  2. Create a bootable USB with Rufus (Windows) or dd (Linux)
  3. Boot from USB
  4. Run the graphical installer
  5. Follow the on-screen prompts
The entire process takes 10-20 minutes depending on your hardware.
Can I dual boot ArttulOS with Windows?
Yes! The Calamares installer automatically detects Windows and sets up dual boot. When you boot your computer, you'll get a menu to choose Windows or ArttulOS. Just make sure to:
  • Back up important Windows data first
  • Disable Windows Fast Startup (in Power Options)
  • Have at least 30GB free space on your drive
Will installing ArttulOS delete my files?
It depends on which installation option you choose:
  • "Erase disk": Yes, this deletes everything on the selected drive
  • "Install alongside": No, this shrinks existing partitions and installs ArttulOS next to them
  • "Manual partitioning": Only affects partitions you explicitly choose
ALWAYS back up important data before any OS installation!
How much disk space does ArttulOS need?
  • Minimum: 20 GB (just the OS)
  • Recommended: 40 GB+ (OS + room for programs and files)
  • Ideal: 120 GB+ (comfortable for most users)
SSD is highly recommended for better performance.
Can I try ArttulOS without installing it?
Yes! Boot from the USB and choose "Try ArttulOS" or "Live Session". This runs ArttulOS from the USB without touching your hard drive. You can test hardware compatibility, explore the interface, and even browse the web. When you're ready to install, double-click "Install ArttulOS" on the desktop.

Software & Package Management

How do I install software on ArttulOS?
Three ways:
  1. Pamac (GUI): Open "Add/Remove Software" from the menu, search for apps, click Install
  2. Pacman (Terminal): sudo pacman -S package-name
  3. AUR Helper (yay/paru): yay -S package-name for AUR packages
Example: sudo pacman -S firefox vlc gimp
What is the AUR and how do I use it?
The Arch User Repository (AUR) is a community-maintained repository with 80,000+ packages not in official repos. It contains things like Google Chrome, Spotify, VS Code, and more.

To use AUR, install an AUR helper: 
sudo pacman -S yay
yay -S google-chrome spotify discord
Or use Pamac GUI which has AUR support built-in (enable in preferences).
How do I update my system?
Terminal: 
sudo pacman -Syu
GUI (Pamac): Open "Add/Remove Software" → Click the updates icon in toolbar

Run updates at least once a week to stay secure and get the latest features.
Can I install Windows apps on ArttulOS?
Some Windows apps work via:
  • Wine: Compatibility layer for Windows programs
  • Proton/Steam: For Windows games on Linux
  • Bottles: Easy Wine prefix manager (GUI)
  • VM (VirtualBox/QEMU): Run Windows in virtual machine
However, most popular Windows software has Linux alternatives or web versions. Ask in our Discord if you need help finding replacements!
Where is Microsoft Office / Adobe Photoshop?
These are Windows/Mac only, but excellent alternatives exist:
  • MS Office → LibreOffice (free), OnlyOffice, or Office 365 web
  • Photoshop → GIMP (free), Krita, or Photopea (web)
  • Illustrator → Inkscape (free)
  • Premiere → Kdenlive, DaVinci Resolve (free)
Most Linux alternatives are free and open source!

Troubleshooting

My Wi-Fi isn't working after installation
First, check if your adapter is detected: 
ip link
If you see your Wi-Fi interface (like wlan0 or wlp3s0), install missing drivers: 
sudo pacman -S linux-firmware
sudo pacman -S broadcom-wl  # For Broadcom cards
For Realtek USB adapters: 
yay -S rtl88xxau-aircrack-dkms-git
Reboot after installing drivers.
Graphics are slow / Screen tearing / Low FPS
Install proper GPU drivers: NVIDIA: 
sudo pacman -S nvidia nvidia-utils
AMD (usually works out of box): 
sudo pacman -S mesa vulkan-radeon
Intel (usually works out of box): 
sudo pacman -S mesa vulkan-intel
Reboot after installing drivers.
How do I fix "Failed to synchronize databases" error?
This usually means mirror or network issues. Try: 
# Refresh mirror list
sudo pacman -Sy archlinux-keyring
sudo pacman -Syu

# If that fails, update mirrors
sudo pacman-mirrors --fasttrack && sudo pacman -Syyu
My system won't boot after update
Boot from the ArttulOS live USB, then:
  1. Mount your root partition: sudo mount /dev/sdXY /mnt
  2. Chroot: sudo arch-chroot /mnt
  3. Reinstall kernel: sudo pacman -S linux linux-headers
  4. Update bootloader: sudo grub-mkconfig -o /boot/grub/grub.cfg
  5. Exit and reboot
Ask in our Discord if you need help!
How do I free up disk space?
Remove package cache: 
sudo pacman -Sc
Remove orphaned packages: 
sudo pacman -Rns $(pacman -Qdtq)
Clean user cache: 
rm -rf ~/.cache/*
Check disk usage: 
ncdu /

Advanced Topics

How do I install a different desktop environment?
ArttulOS comes with Cinnamon, but you can install others: KDE Plasma: 
sudo pacman -S plasma-meta sddm
sudo systemctl enable sddm
GNOME: 
sudo pacman -S gnome gdm
sudo systemctl enable gdm
XFCE: 
sudo pacman -S xfce4 xfce4-goodies lightdm
sudo systemctl enable lightdm
Log out and select your new DE at the login screen.
Can I use ArttulOS as a server?
Yes, but we recommend installing Arch Linux Server edition instead. ArttulOS includes a desktop environment which is unnecessary overhead for servers. However, you can:
  • Disable the display manager: sudo systemctl disable lightdm
  • Boot to multi-user target: sudo systemctl set-default multi-user.target
  • Install server software: sudo pacman -S nginx postgresql docker
How do I contribute to ArttulOS development?
We love contributors! Here's how:
  • Code: Fork our GitHub repos and submit pull requests
  • Bug reports: Open issues on GitHub with detailed info
  • Documentation: Help improve our wiki and guides
  • Community: Help others in Discord/forums
  • Testing: Test development ISOs and report bugs
Join our Discord to get started!
Is ArttulOS secure?
Yes! Security features:
  • Rolling release = immediate security patches
  • Regular kernel updates with latest security fixes
  • Minimal pre-installed software = smaller attack surface
  • Community-audited code (open source)
  • Firewall (ufw) available: sudo pacman -S ufw && sudo ufw enable
Keep your system updated regularly for best security.
